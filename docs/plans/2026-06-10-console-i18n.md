# Console i18n + Leads-tabs-in-header — Plan

> subagent-driven. Groups: A (i18n core + ConsoleLayout header + _layout wiring + parity check),
> B (Leads: tabs→header + translate), C (translate remaining screens: Sidebar, Listings/agency,
> Website, Settings, Login, law). Each group: implement → spec review → quality review.

**Goal:** Bilingual console (EN + "Crnogorski / Srpski") with a remembered header language
switcher; Leads Tours/Inquiries/Clicks tabs in the top header bar; every console string via `t()`.

**Setup:** `export NVM_DIR="$HOME/.nvm"; . "$NVM_DIR/nvm.sh"; nvm use 22`. Console is web-only
(Expo Router / RN-Web). No unit-test runner — verify via typecheck + `expo export` + the parity
check. Reuse: `ConsoleLayout`/`Sidebar` (components), `useAuth` (lib/auth), `groupPhoneClicks`
(lib/clicks), `theme/tokens`.

---

## Task 1 — i18n core (lib/i18n)
**Files:** create `apps/app/lib/i18n/dict.ts`, `apps/app/lib/i18n/index.tsx`,
`apps/app/components/LangSwitcher.tsx`, `apps/app/lib/i18n/__check__/parity.ts`; modify
`apps/app/app/_layout.tsx`; modify root `package.json` (add `check:i18n` script).
- `dict.ts`: `export type Lang = "en" | "sr"; export const LANGS: {code: Lang; label: string}[] =
  [{code:"en",label:"EN"},{code:"sr",label:"ME/SR"}]; export const dict: Record<Lang, Record<string,string>> = { en: {...}, sr: {...} };`
  Seed with `common.*` keys (save, cancel, loading, retry, delete, edit, etc.) — screen keys are
  added by Groups B/C. Pure module, no imports.
- `index.tsx`: `I18nProvider` (context with `lang`,`setLang`), reads `localStorage.kluche_console_lang`
  on mount (guarded), defaults `en`; `useT()` → `{ t, lang, setLang }`; `t(key, vars?)` =
  `(dict[lang][key] ?? dict.en[key] ?? key)` then replace `{var}` tokens from `vars`.
- `LangSwitcher.tsx`: renders `LANGS` as pill toggles, active = current `lang`, onPress `setLang`.
- `_layout.tsx`: wrap the navigator tree in `<I18nProvider>`.
- `parity.ts`: import `dict`; compute key sets for `en`/`sr`; if they differ, `console.error` the
  diff and `process.exit(1)`; else print OK. Add root script `"check:i18n": "tsx apps/app/lib/i18n/__check__/parity.ts"`.
- Verify: `pnpm check:i18n` passes; `pnpm --filter @kluche/app typecheck` green.
- Commit `feat(app): i18n core (en + sr) with persisted language switcher`.
